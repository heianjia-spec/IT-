# 自定义表单模版使用说明

## 概述

表单模版功能允许管理员为不同的资产分类配置**自定义的新增/编辑表单**。每个分类可以拥有独立的表单布局，包括字段的显示、顺序、必填项、控件类型等。

### 核心特性

- **按分类配置**：每个资产分类可配置独立的表单模版
- **树形继承**：子分类自动继承父分类的表单配置，无需重复配置
- **字段复用**：系统内置 41 个标准字段（CPU型号、内存、操作系统等），可直接选用
- **自定义字段**：支持手动输入不在注册表中的自定义字段名
- **区块分组**：表单字段可按区块分组（如"基本信息"、"打印机参数"），支持拖拽排序

---

## 页面入口

左侧菜单 → **设置** → **表单模版管理**，或直接访问 `/settings/template-editor`。

---

## 界面说明

页面分为左右两栏：

| 区域 | 说明 |
|------|------|
| **左侧 - 资产分类树** | 显示所有资产分类的树形结构，点击选择要编辑的分类 |
| **右侧 - 模版编辑器** | 当前选中分类的表单模版配置区 |

分类树中每个节点旁的状态标记：
- <span style="color:#67C23A">**已配置**</span>：该分类有自己的自定义模版
- <span style="color:#909399">**继承**</span>：该分类使用继承自父分类的模版

---

## 操作指南

### 1. 为分类创建自定义模版

1. 在左侧分类树中**点击**目标分类（如"打印机"）
2. 右侧会显示当前生效的表单配置
3. 如果显示"当前使用继承模版"，点击 **"创建自定义模板"** 按钮
4. 系统会预填继承的模板内容，你可以在此基础上修改

### 2. 添加区块

1. 在模版编辑器底部点击 **"+ 添加区块"**
2. 输入区块名称（如"打印机参数"）
3. 可以通过 **上移/下移** 调整区块顺序

### 3. 添加字段

1. 在目标区块内点击 **"+ 添加字段"**
2. 在弹出的对话框中配置字段：

| 配置项 | 说明 | 示例 |
|--------|------|------|
| **字段名(key)** | 字段的唯一标识 | 可从下拉列表选择系统内置字段，或直接输入自定义字段名 |
| **字段标签** | 表单中显示的字段标题 | 选择内置字段时自动填充 |
| **控件类型** | 输入控件：文本/数字/多行文本/日期/下拉选择/开关/树形选择 | 选择内置字段时自动匹配 |
| **列宽(span)** | 字段在表单中的宽度（6-24） | 8=1/3行, 12=半行, 24=整行 |
| **排序** | 字段在区块中的显示顺序 | 数字越小越靠前 |
| **是否必填** | 是否强制要求填写 | |
| **占位提示** | 输入框的 placeholder 文字 | "请输入打印机型号" |
| **帮助文本** | 字段下方的提示信息 | "网络打印机需填写IP地址" |
| **选项(select)** | 当控件类型为"下拉选择"时，定义选项列表 | 每行一个，格式：`value=label` |
| **条件显示** | 根据其他字段的值决定是否显示此字段 | 如"连接方式=有线网络时显示IP地址" |

4. 点击 **"确定"** 保存字段

### 4. 编辑/删除字段

- 点击字段行的 **"编辑"** 按钮修改字段配置
- 点击 **"↑"/"↓"** 调整字段排序
- 点击 **"删除"** 移除字段

### 5. 条件显示配置

条件显示允许某个字段在满足条件时才显示，减少表单冗余信息。例如：

| 场景 | 字段 | 条件 |
|------|------|------|
| 网络打印机才需要IP地址 | IP地址 | `connect_type = wired` |
| 彩色打印机才显示色彩配置 | 是否彩色 | `printer_type = laser` |

配置方式：在字段编辑对话框中填写：
1. **字段名**：依赖的字段 key
2. **操作符**：`等于(eq)` / `不等于(not_eq)` / `包含(in)` / `非空(not_null)`
3. **值**：触发条件的目标值（非空操作符不需要填写值）

### 6. 保存模版

编辑完成后，点击右上角 **"创建自定义模板"** 或 **"更新模板"** 按钮保存。

## 继承机制说明

### 继承规则

1. 子分类**自动继承**最近祖先分类的自定义模版
2. 同名区块中的同名字段（key 相同），子分类覆盖父分类
3. 子分类可以**添加新的区块和字段**，不影响父分类

### 继承链

选中分类后，顶部会显示继承链，例如：

> **硬件（模版）** → 打印机 → 激光打印机

表示"激光打印机"继承自"打印机"，而"打印机"有自己的模版。如果"打印机"有自定义模版，链中会标注"(模版)"。

### 恢复继承

如果某个分类已经有自定义模版，想恢复使用父分类的模版：
1. 选中该分类
2. 点击 **"删除自定义模板（恢复继承）"** 按钮
3. 确认删除

## 内置字段注册表

系统预置了 41 个标准字段，按类别分组：

| 类别 | 字段示例 |
|------|---------|
| **硬件通用** | CPU型号、内存、硬盘、操作系统、IP地址、MAC地址、网卡信息、机架位置 |
| **打印机** | 打印机类型、是否彩色、是否双面、连接方式、硒鼓/墨盒型号 |
| **网络设备** | 网络设备类型、端口类型、PoE支持、堆叠配置 |
| **软件** | 软件类型、版本号、授权类型、许可证号、许可到期 |
| **耗材** | 耗材类型、单位、当前库存、最低库存预警、适用设备、单价 |

## 示例：打印机模版配置

已为**打印机**（`hw_printer`）和**打印机耗材**（`cn_print`）创建了模版。

### 打印机模版（2个区块，19个字段）

```
┌─ 基本信息（11个字段） ─────────────────────────┐
│ 资产名称*  资产编号  状态*  所属部门  存放位置  │
│ 序列号  采购日期  价格  供应商  保修截止  备注  │
└───────────────────────────────────────────────┘
┌─ 打印机参数（8个字段）─────────────────────────────┐
│ 打印机类型*  是否彩色  是否双面  连接方式          │
│ 硒鼓/墨盒型号  IP地址  MAC地址  固件版本           │
└──────────────────────────────────────────────────┘
```

### 打印机耗材模版（2个区块，16个字段）

```
┌─ 基本信息（9个字段） ─────────────────────────┐
│ 耗材名称*  资产编号  状态*  存放位置           │
│ 批次号  采购日期  总价  供应商  备注           │
└───────────────────────────────────────────────┘
┌─ 耗材参数（7个字段）──────────────────────────┐
│ 耗材类型*  单位*  单价  当前库存               │
│ 最低库存预警  适用设备  上次采购日期            │
└───────────────────────────────────────────────┘
```

## 注意事项

1. **字段名(key) 必须唯一**：同一分类下的表单中，字段 key 不能重复
2. **自定义字段**：使用注册表之外的字段名时，需要在资产模型中存在对应字段才能正确存储数据
3. **模版不影响已有数据**：修改模版只影响新增/编辑资产的表单展示，不会修改已有资产数据
4. **父分类模版修改**：修改父分类的模版会影响所有继承它的子分类
5. **浏览器兼容**：建议使用 Chrome/Firefox 最新版本
